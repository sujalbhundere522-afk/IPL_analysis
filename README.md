# IPL Performance Analysis Dashboard

## Project Overview
This project analyzes IPL performance using Power BI and DAX.

## Features
- Champion & Runner-Up Analysis
- Orange Cap & Purple Cap Statistics
- Most Fours & Most Sixes Analysis
- Team Performance Dashboard
- Player Performance Dashboard
- Season-wise Filtering

## Tools Used
- Power BI
- DAX
- GitHub

## Datasets
- ball_by_ball_data.csv
- ipl_matches_data.csv
- players-data-updated.csv
- teams_data.csv
